# Freedom World Brand Specification
## Extracted from info.freedom.world

### Colors
- Background: #050314 (deep dark purple-black)
- Background alt: #0B1036 (dark navy)
- Card bg: rgba(255,255,255,0.05) - glassmorphism
- Card bg hover: rgba(255,255,255,0.1)
- Primary accent (green): #10F48B (rgb 16,244,139)
- Blue accent: #1248C8 (rgb 18,72,190)
- Blue gradient: #0E3285 → #1245C8
- Pink/magenta accent: #F742A2 (used in gradients)
- Cyan accent: #36BBF6 / #37C8F5
- Text primary: #F4F4FC (near-white)
- Text secondary: #A6A7B5 (light gray)
- Text tertiary: #67697C (medium gray)
- Card border: rgba(255,255,255,0.1)

### Gradients
- Hero glow: linear-gradient(-70deg, rgba(247,66,162,0.25), rgba(247,66,162,0.33), rgba(54,187,246,0.6))
- Blue card: linear-gradient(117deg, rgba(24,75,255,0.2), rgba(23,74,255,0.3))
- Feature accent: linear-gradient(117deg, rgba(0,194,255,0), rgb(255,41,195))
- Fade to black: linear-gradient(0deg, rgb(2,2,16), rgba(2,2,16,0))

### Typography
- Font: "Encode Sans Expanded", sans-serif
- For Thai: Need Thai-compatible font — Noto Sans Thai or similar
- Headings: Bold/Extra-bold, large sizes, uppercase for section labels
- Body: Regular weight, good line height

### Design Patterns
- Glassmorphism cards with subtle border and backdrop blur
- Dark theme throughout
- Generous whitespace
- Rounded corners (16px+ on cards)
- Subtle gradient overlays and glows
- Section labels in small caps/uppercase with accent color
- Feature cards with icon + title + description
- Full-bleed hero with overlay gradient
